# 软件的安装及部署

用户如果使用本地的单台工作站进行测试，则Foretify的相关软件及License服务器也可部署在该机器上。需要安装部署的组件包括：

- Foretify License Server: 用于提供license服务
- Foretify平台：
  - Foretify Developer
  - Foretify Manager

!!! note "注意"
    Foretify V-Suites随Foretify Developer一并安装

### 硬件及系统需求

推荐配置：
- CPU cores: 16
- RAM: 32 GB 
- STORAGE: 100GB
- Operation System: Ubuntu 20.04/22.04
